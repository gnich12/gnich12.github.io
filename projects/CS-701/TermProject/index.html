About:

This app is a term from project for BU MSCIS-701 course

I use Html5 and JQuery/UI to program all the behavior of this application, such as the hide/show and the events when user clicks the buttons etc. I also use an api to pick random words for the word section of the game. As well Im embedding some videos from youtube as a main prize.

How To:

In order to find the code, You have two options:

  1. Find the 3 color code, by clicking the buttons for the right combination
  2. Find the correct word by drag in the letters and put the in the correct order to find the secret word
